# Greek Tavern Menu

This is a small project I made using **HTML and CSS** — a digital menu for a traditional Greek tavern Laterna.  
It’s a simple, static web page that focuses on clean layout, readability, and a bit of Greek character.

---

## About

I built this to practice front-end design and experiment with styling and layout.  
The menu includes different sections like appetizers, main dishes, desserts, and drinks, all styled with basic CSS.

---

## Technologies Used

- HTML5  
- CSS3  

No frameworks or JavaScript — just pure HTML and CSS.
